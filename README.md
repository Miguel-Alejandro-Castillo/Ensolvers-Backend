####Requirements
* MySQL 5.7
* Spring Boot 2.4
* Apache Maven 3.6.0
* Java openjdk 8
* Intellij Idea(Optional)

####commands to run the application(see script up.sh)
1. chmod +x up.sh
2. ./up.sh <datasource.username> <datasource.password> <datasource.url>

   for example,
   ./up.sh root root jdbc:mysql://localhost:3306/ensolvers


Enjoy!
